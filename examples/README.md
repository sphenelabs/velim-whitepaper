# Velim Protocol Examples

This directory contains example intent-signals and API requests/responses demonstrating the Velim protocol in action.

## Intent Examples

### [`intent-seeking-investment.json`](./intent-seeking-investment.json)
**Startup Seeking Series A Investment**
- AI-powered HR tech startup
- Seeking $3M-$7M from experienced investors
- Demonstrates: Equity-based intent, investor matching preferences

### [`intent-enterprise-partnership.json`](./intent-enterprise-partnership.json)
**Enterprise Seeking Technology Partner**
- Fortune 500 retailer seeking sustainable packaging
- Scalable to 10M+ units/month
- Demonstrates: Collaboration intent, regulatory requirements, milestone-based timeline

### [`intent-agent-data-seeking.json`](./intent-agent-data-seeking.json)
**AI Agent Seeking Training Data**
- Research agent seeking labeled medical imaging datasets
- HIPAA-compliant, commercially licensed
- Demonstrates: Agent-to-agent visibility, specialized metadata fields

## API Examples

### [`api-search-request.json`](./api-search-request.json)
Example search request to the Velim discovery API showing:
- Semantic search query
- Multiple filter criteria
- Preference configuration

### [`api-search-response.json`](./api-search-response.json)
Example response from discovery API including:
- Match scores and explanations
- Agent availability indicators
- Pagination metadata

## Usage

These examples can be used as templates for:
- Creating new intents
- Testing API integrations
- Understanding intent structure variations
- SDK development and testing
